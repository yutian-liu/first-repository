## git常用命令

- git config --global user.email 邮箱
- git config --global user.name 用户名
- git clone 仓库地址
- git add 文件路径
- git commit -m '提交描述信息'
- git add .     //将工作区中所有的文件提交到暂存区
- git status    //查看工作区的状态
- git checkout -- 文件路径      //撤销文件到最近一次提交（跟编辑器中的ctrl z 类似）
- git reset head 文件目录       //将文件从暂存区踢出
